# Dependências
Para utilizar deve-se instalar:
```
pip install matplotlib
```
```
pip install mpmath
```
```
pip install pandas
```
```
pip install scipy
```
Todos de uma vez:
```
pip install matplotlib mpmath pandas scipy
```
